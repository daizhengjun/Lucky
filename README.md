## 技术栈

vue2 + vuex + vue-router + webpack + ES6/7 + less + element-ui + electron-vue

## 项目运行
```
git clone https://github.com/daizhengjun/Lucky.git

cd lucky

npm install 或 yarn(推荐)

npm run electron:serve
```
